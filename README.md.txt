# Nested Lookup Telegram Bot

A Telegram bot for searching and modifying nested JSON data.

## Commands

| Command | Description |
|---------|-------------|
| `/start` | Welcome message |
| `/lookup <key>` | Find all values for a key |
| `/keys` | Show all unique keys |
| `/view` | View your document |
| `/update <key> <value>` | Update all occurrences of a key |
| `/delete <key>` | Delete all occurrences of a key |
| `/reset` | Reset to default document |

## Deployment

This bot is deployed on Render.com.
